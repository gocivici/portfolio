Trinteract: an Open-source 3D Input Device

I developed Trinteract as part of my master's degree project. It‚Äôs an open-source Arduino-compatible universal 3DOF input device. In other words, a joystick capable of moving in all 3 Dimensions.

Trinteract is attached to a computer and can be used to navigate 3D applications like Blender or Fusion. It can also be used as a midi input device for music production software like Ableton. Additionally, third-party tools like KeySticks can be used to map the input for apps that do not provide support 3DOF devices.

Why?

This project started with a narrow topic, focusing on the challenges designers face while navigating 3D graphics applications. After conducting observations and interviews, it was evident that traditional input devices like the keyboard and mouse were not capable of providing a comfortable navigation experience for the users as they required a combination of inputs for navigating or orienting the object. These combinations were mostly in the form of holding a key on the keyboard and moving the mouse in a certain way.

How?

Trinteract uses a custom PCB housing a Triple-Axis Magnetometer and a Neodymium magnet to detect motion in 3D space. It comes with a 3D-printed knob but electromagnets, magnetic textiles, or any magnetic object can also be used to control the input, creating more opportunities for interaction.
